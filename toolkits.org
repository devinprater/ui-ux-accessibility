#+title: UI Toolkits

A toolkit, here, refers to a set of widgets and UI tools to make
graphical interfaces. These may be for local or web applications. The
following is a list of toolkits that provide accessible interfaces.

- [[https://github.com/AccessKit/accesskit][AccessKit]]
- [[https://beeware.org][BeeWare]]
- [[https://docs.wxwidgets.org/3.0/classwx_accessible.html][WX-Widgets (Python)]]
